## مقدمة

لوحة التحكم الإدارية هي مركز إدارة منصة "ثانوي" التعليمية. تتيح لك:

- إدارة المستخدمين والطلاب
- إضافة وتحرير المواد والدورات
- إنشاء وإدارة الاختبارات
- مراقبة الأداء والإحصائيات
- إرسال الإشعارات للمستخدمين
- إدارة الصلاحيات والأدوار

---

## تسجيل الدخول

### خطوات الدخول

1. افتح المتصفح وانتقل إلى: `https://admin.thanawy.com`
2. أدخل بريدك الإلكتروني وكلمة المرور
3. إذا كنت قد فعلت المصادقة الثنائية (2FA)، أدخل الرمز
4. اضغط على "تسجيل الدخول"

### استعادة كلمة المرور

1. انقر على "نسيت كلمة المرور؟"
2. أدخل بريدك الإلكتروني
3. افتح البريد واضغط على رابط إعادة التعيين
4. اكتب كلمة مرور جديدة

## لوحة التحكم الرئيسية

### المكونات الرئيسية

| القسم | الوصف |
|-------|-------|
| **إحصائيات سريعة** | عرض عدد المستخدمين، الاختبارات، الإيرادات |
| **الرسوم البيانية** | تحليلات مرئية للنمو والأداء |
| **الإشعارات الأخيرة** | آخر التنبيهات والتحديثات |
| **المهام العاجلة** | مهام تتطلب اهتمامك الفوري |

### تخصيص لوحة التحكم

يمكنك تخصيص ترتيب الأقسام:

1. اضغط على زر "تخصيص" في أعلى الصفحة
2. اسحب الأقساط وأسقطها بالترتيب المفضل
3. اضغط "حفظ"

## إدارة المستخدمين

### عرض قائمة المستخدمين

- انتقل إلى **المستخدمين** من القائمة الجانبية
- استخدم شريط البحث للبحث بالاسم أو البريد
- استخدم الفلاتر لتصفية حسب الدور أو الحالة

### إضافة مستخدم جديد

1. اضغط زر **"مستخدم جديد"**
2. املأ النموذج:
   - الاسم الكامل
   - البريد الإلكتروني
   - الدور (طالب، معلم، مشرف)
   - الصف الدراسي (للطلاب)
3. اضغط **"حفظ"**

> **ملاحظة:** سيتلقى المستخدم بريداً تلقائياً لتفعيل حسابه

### تعديل مستخدم

1. اضغط على صف المستخدم
2. اضغط أيقونة **التعديل** ✏️
3. عدل البيانات
4. اضغط **"حفظ"**

### تعطيل/تفعيل حساب

1. افتح صفحة المستخدم
2. اضغط على مفتاح **"الحالة"**
3. اختر "نشط" أو "معطل"

### تصدير قائمة المستخدمين

1. اضغط زر **"تصدير"**
2. اختر التنسيق (Excel, CSV, PDF)
3. اختر الحقق المراد تصديرها
4. اضغط **"تصدير"**

## إدارة المواد والدورات

### إضافة مادة جديدة

1. انتقل إلى **المواد** > **إضافة مادة**
2. املأ المعلومات:
   - اسم المادة
   - الوصف
   - الصف الدراسي
   - صورة الغلاف
3. اضغط **"حفظ"**

### إضافة محتوى للمادة

1. افتح صفحة المادة
2. اضغط **"إضافة درس"**
3. اختر نوع المحتوى:
   - فيديو
   - ملف PDF
   - نص تفاعلي
   - اختبار
4. ارفع الملف أو أدخل الرابط
5. اضغط **"نشر"**

### ترتيب الدروس

1. افتح صفحة المادة
2. اضغط **"ترتيب"**
3. اسحب الدروس بالترتيب المطلوب
4. اضغط **"حفظ"**

## إدارة الاختبارات

### إنشاء اختبار جديد

1. انتقل إلى **الاختبارات** > **اختبار جديد**
2. أدخل المعلومات الأساسية:
   - اسم الاختبار
   - المادة
   - مدة الاختبار (بالدقائق)
   - درجة النجاح
3. اضغط **"متابعة"**

### إضافة أسئلة

**أنواع الأسئلة المدعومة:**

| النوع | الوصف |
|-------|-------|
| **اختيار من متعدد** | سؤال مع 2-4 خيارات |
| **صح/خطأ** | سؤال ثنائي الإجابة |
| **مقالي** | إجابة نصية قصيرة |
| **ملء الفراغات** | إكمال جملة |

**خطوات الإضافة:**

1. اختر نوع السؤال
2. اكتب نص السؤال
3. أدخل الخيارات (للاختيار المتعدد)
4. حدد الإجابة الصحيحة
5. أدخل الدرجة (افتراضي: 1)
6. اضغط **"حفظ السؤال"**

### معاينة الاختبار

1. من صفحة الاختبار، اضغط **"معاينة"**
2. جرب الاختبار كما يراه الطالب
3. عدل إذا لزم الأمر

### نشر الاختبار

1. اضغط **"نشر"**
2. اختر:
   - **نشر فوري:** متاح مباشرة
   - **جدولة:** حدد التاريخ والوقت
3. اضغط **"تأكيد"**

## الإشعارات والبث

### إرسال إشعار فردي

1. افتح صفحة المستخدم
2. اضغط **"إرسال إشعار"**
3. اكتب العنوان والرسالة
4. اضغط **"إرسال"**

### إرسال بث جماعي

1. انتقل إلى **الإشعارات** > **بث جديد**
2. اختر المستلمين:
   - جميع المستخدمين
   - فئة محددة (مثلاً: طلاب ثالثة ثانوي)
   - مستخدمين محددين
3. اكتب عنوان الرسالة
4. اكتب المحتوى
5. اختر القنوات:
   - داخل التطبيق
   - البريد الإلكتروني
   - رسالة نصية
6. اضغط **"إرسال"**

### جدولة إشعار

1. في نموذج الإشعار، اضغط **"جدولة"**
2. اختر التاريخ والوقت
3. اضغط **"تأكيد"**

## التقارير والتحليلات

### عرض التقارير الجاهزة

| التقرير | الوصف |
|---------|-------|
| **نمو المستخدمين** | إحصائيات التسجيل الجديد |
| **أداء الاختبارات** | متوسط الدرجات والنجاح |
| **الإيرادات** | المدفوعات والاشتراكات |
| **نشاط المستخدمين** | الدخول والتفاعل |

### إنشاء تقرير مخصص

1. انتقل إلى **التقارير** > **تقرير جديد**
2. اختر نوع التقرير:
   - جدول
   - رسم بياني (خطي، أعمدة، دائري)
3. اختر البيانات المطلوبة
4. اضبط الفلاتر والتواريخ
5. اضغط **"إنشاء"**

### تصدير التقرير

1. افتح التقرير
2. اضغط **"تصدير"**
3. اختر التنسيق
4. اضغط **"تحميل"**

## الإعدادات

### إعدادات الحساب

- **تغيير كلمة المرور:** كل 90 يوماً يُنصح بتغييرها
- **تفعيل 2FA:** مصادقة ثنائية للأمان
- **تعديل الملف الشخصي:** الاسم، الصورة، البريد

### إعدادات النظام

- **الثيم:** فاتح / داكن / تلقائي
- **اللغة:** العربية / الإنجليزية
- **التنبيهات:** تفعيل/تعطيل الإشعارات

### إدارة الصلاحيات

1. انتقل إلى **الإعدادات** > **الصلاحيات**
2. اختر الدور (مشرف، معلم، محرر)
3. حدد الصلاحيات:
   - عرض المستخدمين
   - تعديل المستخدمين
   - حذف المستخدمين
   - إدارة المواد
   - إدارة الاختبارات
4. اضغط **"حفظ"**

## اختصارات لوحة المفاتيح

### اختصارات عامة

| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl + K` | فتح البحث السريع |
| `Ctrl + /` | فتح لوحة الأوامر |
| `Ctrl + B` | تبديل الشريط الجانبي |
| `Ctrl + H` | الذهاب للرئيسية |
| `Alt + 1-9` | الانتقال للقسم (1-9) |
| `Esc` | إغلاق القائمة المفتوحة |

### اختصارات التنقل

| الاختصار | الوظيفة |
|----------|---------|
| `↑ / ↓` | التنقل بين العناصر |
| `Enter` | فتح/اختيار العنصر |
| `Tab` | الانتقال للحقل التالي |

### اختصارات الإجراءات

| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl + N` | إنشاء جديد |
| `Ctrl + S` | حفظ |
| `Ctrl + D` | حذف |
| `Ctrl + E` | تعديل |

### اختصارات سريعة

| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl + Shift + L` | تبديل الوضع الليلي |
| `Ctrl + Shift + ?` | عرض الاختصارات |

## نصائح وأفضل الممارسات

### للأمان 🔒

1. **فعّل المصادقة الثنائية (2FA)**
2. **لا تشارك بيانات الدخول** مع أي شخص
3. **غيّر كلمة المرور** كل 3 أشهر
4. **اخرج من الجلسات** عند الانتهاء من العمل

### للأداء ⚡

1. **أغلق الأقسام غير المستخدمة** لتوفير الذاكرة
2. **استخدم الفلاتر** بدلاً من تحميل كل البيانات
3. **صدر التقارير الكبيرة** بدلاً من عرضها في المتصفح

### للإنتاجية 🚀

1. **استخدم الاختصارات** لتوفير الوقت
2. **خصّص لوحة التحكم** حسب أولوياتك
3. **فعّل الإشعارات** للأحداث المهمة فقط
