# نظام إدارة الوقت المتطور

## نظرة عامة

تم تطوير نظام شامل لإدارة الوقت يتضمن ميزات متقدمة للتحليل والإنتاجية والإشعارات.

## الميزات الجديدة

### 1. التحليلات المتقدمة (TimeAnalytics)
- **توزيع وقت المذاكرة**: عرض توزيع ساعات المذاكرة خلال الأسبوع
- **اتجاه إكمال المهام**: مخطط بياني لإكمال المهام خلال آخر 7 أيام
- **توزيع الوقت حسب المادة**: تحليل الوقت المخصص لكل مادة دراسية
- **ساعة الذروة**: تحديد أفضل ساعة للمذاكرة بناءً على البيانات
- **درجة الإنتاجية**: حساب درجة إنتاجية شاملة (0-100%)
- **متوسط الجلسة**: حساب متوسط مدة جلسات المذاكرة

### 2. رؤى الإنتاجية (ProductivityInsights)
- **توصيات ذكية**: توصيات مخصصة بناءً على أنماط المستخدم
- **تنبيهات المهام المتأخرة**: إشعارات فورية للمهام المتأخرة
- **مواعيد قريبة**: تنبيهات للمهام القريبة من الموعد النهائي
- **مهام عالية الأولوية**: عرض المهام التي تحتاج انتباه فوري
- **انتظام المذاكرة**: حساب نسبة الانتظام خلال آخر 7 أيام
- **دقة التقدير**: تقييم دقة تقدير الوقت المطلوب للمهام
- **أفضل وقت للمذاكرة**: تحديد الساعة الأكثر إنتاجية
- **مجالات التركيز**: عرض المواد التي تحتاج إلى مزيد من الاهتمام

### 3. وظائف التصدير (ExportDialog)
- **تصدير CSV**: تصدير البيانات بصيغة CSV للتحليل
- **تصدير JSON**: تصدير البيانات بصيغة JSON للمعالجة البرمجية
- **تصدير PDF/HTML**: إنشاء تقارير HTML قابلة للطباعة
- **اختيار الفترة**: تصدير أسبوع أو شهر أو جميع البيانات
- **ملخص البيانات**: عرض ملخص قبل التصدير

### 4. اختصارات لوحة المفاتيح
- **Ctrl/Cmd + 1-6**: التبديل بين علامات التبويب
- **Ctrl/Cmd + T**: بدء/إيقاف المؤقت
- **Ctrl/Cmd + R**: تحديث البيانات
- **Ctrl/Cmd + N**: فتح تبويب المهام (مهمة جديدة)
- **Ctrl/Cmd + Shift + N**: فتح تبويب التذكيرات (تذكير جديد)
- **Esc**: إغلاق النوافذ المنبثقة

### 5. إشعارات المهام المتأخرة (useOverdueNotifications)
- **فحص دوري**: فحص المهام كل دقيقة
- **إشعارات فورية**: إشعارات للمهام المتأخرة أكثر من 24 ساعة
- **تنبيهات المواعيد القريبة**: إشعارات للمهام القريبة من الموعد (3 ساعات)

### 6. تحسين معالجة الأخطاء
- **Promise.allSettled**: معالجة أفضل للأخطاء تسمح بالنجاح الجزئي
- **معالجة الأخطاء الفردية**: معالجة منفصلة لكل مصدر بيانات
- **رسائل خطأ واضحة**: رسائل خطأ مفصلة ومفيدة للمستخدم
- **الاستمرار عند الفشل الجزئي**: استمرار العمل حتى لو فشل بعض المصادر

## المكونات الجديدة

### TimeAnalytics.tsx
مكون شامل لعرض التحليلات المتقدمة والإحصائيات.

### ProductivityInsights.tsx
مكون لعرض الرؤى والتوصيات الذكية.

### ExportDialog.tsx
مكون حوار لتصدير البيانات بصيغ مختلفة.

### KeyboardShortcutsHelp.tsx
مكون مساعدة لعرض اختصارات لوحة المفاتيح.

## الـ Hooks الجديدة

### useKeyboardShortcuts
Hook لإدارة اختصارات لوحة المفاتيح.

### useOverdueNotifications
Hook لمراقبة المهام المتأخرة وإرسال الإشعارات.

## ملفات المساعدة

### exportUtils.ts
وظائف مساعدة لتصدير البيانات:
- `exportToCSV()`: تصدير إلى CSV
- `exportToJSON()`: تصدير إلى JSON
- `generatePDFReport()`: إنشاء تقرير HTML
- `printPDFReport()`: طباعة التقرير
- `downloadFile()`: تحميل الملف

## الاستخدام

### إظهار التحليلات
```tsx
// في صفحة Dashboard
<Button onClick={() => setShowAnalytics(!showAnalytics)}>
  {showAnalytics ? 'إخفاء التحليلات' : 'إظهار التحليلات'}
</Button>
```

### تصدير البيانات
```tsx
<ExportDialog 
  tasks={tasksLocal}
  studySessions={studySessionsLocal}
  reminders={remindersLocal}
/>
```

### استخدام الاختصارات
الاختصارات تعمل تلقائياً عند تحميل الصفحة. استخدم `KeyboardShortcutsHelp` لعرض قائمة الاختصارات.

## التكامل مع النظام

- **ErrorManager**: معالجة الأخطاء المتكاملة
- **Notification System**: إشعارات متكاملة مع نظام الإشعارات
- **Toast Notifications**: إشعارات منبثقة للمهام المتأخرة

## الأداء

- **استخدام useMemo**: تحسين الأداء للعمليات الحسابية المعقدة
- **Promise.allSettled**: تحميل متوازي مع معالجة أخطاء محسنة
- **Lazy Loading**: تحميل المكونات عند الحاجة فقط
