## 📊 AUDIT OUTPUT FORMAT

### Critical Issues (🔴)
- **Issue**: Specific problem
- **Location**: `file:line_number` where applicable
- **Root Cause**: Why this happened
- **Impact**: Business/technical consequence
- **Likelihood**: Probability of occurring
- **Example**: Code snippet showing the problem
- **Fix Priority**: Timeline for fixing
- **Suggested Fix**: Concrete solution

### Major Issues (🟠)
- Format same as critical, but lower impact

### Minor Issues (🟡)
- Format same, but cosmetic or small impact
